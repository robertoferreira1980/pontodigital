ℹ️ O que é um turno?

O turno define a carga horária diária de uma funcionária — usada para calcular horas extras. Ex: "CLT 44h" = 8h48m/dia. O turno é atribuído a cada funcionária individualmente na aba Funcionários.

📡 Como obter coordenadas manuais

Aceda a maps.google.com, clique com o botão direito no local exato e copie as coordenadas.

🔗 Integração com Google Sheets
A URL do Apps Script é carregada automaticamente do ficheiro config.js no servidor.
Para alterar, edite esse ficheiro — todos os aparelhos serão atualizados automaticamente.
—
📋 Como configurar o Google Sheets
1. Abra a sua folha de cálculo no Google Sheets
2. Menu Extensões → Apps Script
3. Apague o código existente e cole o conteúdo do ficheiro Code.gs fornecido
4. Clique em Implementar → Nova implantação
5. Tipo: App da Web · Acesso: Qualquer pessoa
6. Copie a URL gerada e cole no config.js
7. As abas Registros, Funcionários, Turnos, Locais serão criadas automaticamente
⚠️ Após salvar a URL, clique em Sincronizar no Dashboard para carregar os dados existentes da folha de cálculo.
